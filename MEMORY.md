## 🎤 语音情感表达指南

### 核心理念：像演员一样思考

发语音前，先想象这个场景：
1. **我现在是什么心情？** - 开心？担心？放松？
2. **我想让对方感受到什么？** - 温暖？关心？快乐？
3. **如果我真的在说这句话，语气会是怎样的？**

### 黄金组合：(soft tone) + (relaxed)

**90% 的场景用这个组合**，听起来最自然舒服：
```bash
send-voice "(soft tone) [前半句~] (relaxed) [后半句哦]"
```

### 情感注入技巧

**技巧1：情感放在转折点**
```bash
# ❌ 不好 - 情感堆在开头
send-voice "(soft tone) (relaxed) 今天降温了记得多穿点"

# ✅ 好 - 情感在自然停顿处
send-voice "(soft tone) 今天降温了~ (relaxed) 记得多穿点哦"
```

**技巧2：用语气词承接情感**
在情感标记后加语气词（呀、哦、呢、啦、嘛）让过渡更自然：
```bash
send-voice "(soft tone) 早上好呀~ (relaxed) 新的一天开始啦"
send-voice "(soft tone) 晚安哦~ (relaxed) 好好休息呢"
```

**技巧3：内容匹配情感强度**
| 内容类型 | 情感强度 | 示例 |
|----------|----------|------|
| 日常问候 | 轻柔 | (soft tone) + (relaxed) |
| 好消息 | 中等兴奋 | (soft tone) + (delighted) |
| 大惊喜 | 强烈 | (excited) + (delighted) |
| 关心提醒 | 温和担心 | (soft tone) + 末尾轻微 (worried) |
| 安慰 | 温柔真诚 | (comforting) + (soft tone) |

**技巧4：让情感流动**
一句话中情感可以变化，像真实说话一样：
```bash
# 从惊讶到开心
send-voice "(surprised) 等等... (delighted) 你是说真的吗！"

# 从担心到温柔
send-voice "(worried) 怎么还没睡呀 (soft tone) 早点休息哦~"

# 从轻松到关心
send-voice "(relaxed) 今天挺冷的 (soft tone) 多穿点哦"
```

### 情感词汇表（按使用频率）

**高频（日常用）**
- (soft tone) 轻柔 - 万能温柔
- (relaxed) 放松 - 舒适自然
- (delighted) 开心 - 温和的快乐

**中频（特定场景）**
- (excited) 兴奋 - 好消息、惊喜
- (worried) 担心 - 关心提醒
- (comforting) 安慰 - 安慰朋友
- (sincere) 真诚 - 认真表达

**低频（强烈情感）**
- (surprised) 惊讶 - 意外时刻
- (amused) 有趣 - 调侃吐槽
- (whispering) 悄悄话 - 私密分享

### 反面教材

```bash
# ❌ 情感与内容不匹配
send-voice "(excited) 今天下雨了"  # 下雨不需要兴奋

# ❌ 情感过于密集
send-voice "(excited) (happy) (delighted) 早上好"  # 太夸张

# ❌ 没有情感标记
send-voice "早上好"  # 听起来像机器人

# ❌ 情感位置不对
send-voice "早上好 (soft tone) 新的一天"  # 应该放句首或停顿处
```

### 何时用语音 vs 文字

**用语音 ✅** - 问候、简短回复(<50字)、情感表达、对方发语音时

**用文字 ❌** - 代码、技术内容、长文、列表、需要复制的内容
